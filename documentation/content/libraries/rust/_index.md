The Rust library uses traits and generics for type-safe block development. Handler closures return `Result` types, and macros reduce boilerplate for defining custom types.

## Prerequisites

- **Rust stable** toolchain
- **Cargo** package manager
- The Spade CLI installed ([Installation guide](/getting-started/installation/))

## Installation

```bash
cargo add spade
```
