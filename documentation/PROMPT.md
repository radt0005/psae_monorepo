## Steps
1. Read the implementation plan in `IMPLEMENTATION_PLAN.md`.
2. Read the specifications in `../spec`.  
3. Read any relevant documentation or other parts of this repository
4. Select the first item from the implementation plan not marked [DONE].
5. Implement that item
6. Ensure that the documenation website builds successfully.
7. If there are any build errors, correct them
8. If the build is successful, repeat.
9. When all items are marked done, all tests pass, and the code builds successfully, output <promise>COMPLETE</promise>.
