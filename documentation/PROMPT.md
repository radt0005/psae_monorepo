# Instructions

This folder contains the implementation of the core library for the spade system. The specifications for the system are located in `../spec/`.  Please consult these to have a thorough understanding of the system.

Your task is to create a detailed implementation plan for the documentation website for the spade system based on the specifications in `../spec/` and code in this repository.  Please format the implementation plan as a markdown checklist.  

This website should include documentation for all of the language-specific libraries in `../libs/`.  These libraries are for developing plugins (blocks) for the spade system.  Please include tutorials and examples for creating blocks.  

Also, please include the documentation for the CLI for the system in `../cli`.  This command line utility enables the creation of new blocks and generally makes the creation and testing of new blocks much easier.  Again, please include a getting started page.  

This application will be at https://docs.spade.app, and the main product page will be at https://spade.app.  The main page will link to this one for the documentation. 

Note that much of the readers of this documentation will be researchers, not developers.  Please make sure that all of the documentation is thorough, free of jargon, and easy to understand.

It should also include information and specifications for writing YAML pipelines.  All of this should be searchable.

After the completion of the implementation plan, this folder should contain a working website that builds successfully.  

The website is based on the `zola` website builder. Please read the documentation as needed.
