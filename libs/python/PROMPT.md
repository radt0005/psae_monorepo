# Implementation Planning

Please create a detailed implementation plan in ./IMPLEMENTATION_PLAN.md with plans for changes to this library.  

The specifications for this software system are located in `../../spec`.  This folder is the python library for the system.  We have some basic elements stubbed out from prototypes, but most of the library work remains to be implemented.  The specification for the library is in `../../spec/libraries.md`; this one should be for the Python programming language.  The other languages will be implemented later. 

The repository is (mostly) a clean slate.  There is some code implemented, but the majority is yet to be done.  Please include testing of the system in the implementation plan.  

When the implementation plan is completed, this folder should contain a working, tested, robust python library for creating Python blocks in the spade system (see the specifications for more information).  


## Using the repository

This library uses `uv` for package management, and pytest for tests.  Please ensure that tests are developed for all the code, and that they run successfully with `uv run pytest`.
